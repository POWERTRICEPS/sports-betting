# User Manual

## Introduction
Welcome to Sports Betting. This guide provides step-by-step instructions to help you install, set up, and use our Sports Betting web application effectively. Our product provides accurate live win probabilities and player prop projections for NBA games that are going on. Anyone from casual NBA enjoyers to professional sports bettors are able to perform analysis throughout each game based on statistics. The design provides a clear and interactive view of the probabilities and statistics in a clean dashboard.
